Демо приложение для UMI.Framework
======

Демонстрационное приложение на UMI.Framework представляет из себя блог.
В приложении реализован функционал:
    - Авторизация
    - Добавления поста и тэгов к нему
    - Форма обратной связи

### УСТАНОВКА
